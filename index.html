提前網上訂票
美泉宮、漁人堡上層塔樓、霍亨薩爾茨堡要塞建議提前網上購票，避免排長龍。65歲以上大多有優惠票，記得帶護照核實年齡。
💶
現金與信用卡並備
匈牙利用福林（HUF），奧地利用歐元（EUR），捷克用克朗（CZK）。CK小鎮部分小店只收現金。信用卡大城市廣泛接受，但建議每人帶約€200現金備用。
💊
帶備常用藥物
腸胃藥、止痛藥、感冒藥、血壓藥等，歐洲藥房較貴且語言可能不通。建議帶足整個旅程用量，並多帶幾天份量以防萬一。
🛡️
旅遊保險
65歲以上出行，旅遊保險非常重要。建議購買包含醫療緊急援助及緊急送返原居地的保險計劃。出發前確認保單已生效。

國際漫遊 / SIM卡
建議購買歐洲旅遊SIM卡（如Airalo），涵蓋匈牙利、奧地利、捷克。方便使用Google Maps導航及緊急聯絡。

好消息：三個國家購物可統一喺布拉格機場退稅！
你哋係從布拉格機場（PRG）離開歐盟，所以三個國家購買的商品，全部可以帶去布拉格機場統一辦理退稅！

• 🇭🇺 布達佩斯買嘅嘢 → 帶去布拉格機場辦退稅 ✅
• 🇦🇹 維也納/薩爾茨堡買嘅嘢 → 帶去布拉格機場辦退稅 ✅
• 🇨🇿 布拉格買嘅嘢 → 直接喺布拉格機場辦退稅 ✅

⚠️ 重要： 所有商品必須保持未開封/未使用狀態，以備海關查驗！

第一步：各城市購物時
• 索取退稅單： 每次消費時主動告知店員需要「Tax Free」，出示護照，索取 Global Blue 或 Planet 退稅單及收據。

• 填寫資料： 確認退稅單填妥英文姓名、護照號碼、住址、Email，建議選擇信用卡退款。

• 妥善保存： 所有單據放在一起，商品不要拆封（以備海關查驗）。

• 分開存放： 建議將三個國家的退稅單分開夾好，方便布拉格機場辦理。
✈️
第二步：布拉格機場統一辦理
流程順序：

1️⃣ 辦理登機： 先去航空公司櫃檯辦理登機並託運行李。若退稅品在託運行李中，務必告知要去海關蓋章。

2️⃣ 海關蓋章 (Customs)： 找到海關辦公室（通常在航廈檢查站附近），將三個國家的退稅單、護照、登機證、商品一併交海關核對，蓋上「海關章」。

3️⃣ 辦理退稅 (Refund)： 取得海關章後，前往 Global Blue 或 Planet 退稅櫃檯，可一次過辦理三個國家的退稅！
💻
第三步：不同退稅系統處理
• 電子退稅 (E-Tax Free)： 有些退稅單顯示「數位出口驗證 (DEV)」，可在自助機器操作，可能不需人工蓋章。

• 若趕不上退稅櫃檯： 可將蓋好海關章的退稅單填妥信用卡號，投入 Global Blue / Planet 專用信箱郵寄退回。
⚠️
關鍵注意事項
• 預留時間： 建議比一般時間提早至少 1 小時到機場，退稅櫃檯有時排隊很久。

• 退稅單效期： 購買後需在 3 個月內完成海關蓋章。你哋 5/8 布達佩斯買，5/18 布拉格機場辦理，相距10日，完全冇問題 ✅

• 千萬不要遺失蓋章： 沒有海關章的退稅單無法退稅。

• 匈牙利特別提示： 匈牙利 VAT 高達 27%（歐盟最高），退稅金額相對豐厚，值得申請！
📌
總結
三個國家買嘅嘢，統統帶去布拉格機場。帶齊所有退稅單 → 海關蓋章 → 一次過辦理退稅，慳時間又方便！

國際精品：維也納最推薦
對於 LV、Gucci、Chanel 等歐系國際精品，維也納通常是最好的選擇：

• 貨源充足： 維也納精品店密集，新款與經典款貨量比布達佩斯或布拉格更豐富。

• 匯率優勢： 歐元計價，唔需要換匯，避免匯差損失。

• Swarovski： 產自奧地利，維也納買款式最多，定價比香港或台灣便宜。

布達佩斯必買推薦
• 🌶️ 辣椒粉 Paprika — 匈牙利最具代表性特產，分甜味（Édes）同辣味（Csípős），輕巧好攜帶。中央市場買最新鮮！

• 🎲 Rubik's Cube 魔術方塊 — 匈牙利人 Ernő Rubik 於1974年發明，喺布達佩斯買最有意義！

• 🏺 Herend 手繪瓷器 — 匈牙利頂級瓷器品牌，每件手工繪製，花紋精美，可訂製配送回家。

• 🧵 手工刺繡 — Kalocsa 或 Matyó 風格刺繡桌布、圍巾，色彩鮮艷，傳統工藝。

• 🍫 Unicum 藥草酒 / Szamos 杏仁糖 — 匈牙利特色食品，超市買比旅遊區便宜得多。
